LESSON 1:JavaScript

Purpose of JavaScript

JavaScript is used to create dynamic and interactive user experiences on websites. It handles user interface rendering, content loading, real-time updates, personalized feeds, video controls, and tracking user activity.

Static vs Dynamic Websites

Static Web Page: A web page that is delivered exactly as stored and does not change unless manually edited.

Example: Photographer's portfolio website.


Dynamic Website: A website that changes content depending on user actions, preferences, or other variables.

Example: Social media platforms.

About JavaScript

  • The most popular and widely used scripting language on the internet.
  • Recognized and supported by all major web browsers.
  • Developed by Brendan Eich at Netscape under the name Mocha, later renamed to LiveScript.
  • Opened to Internet Explorer in 1996.
  • A trademark of the Oracle Corporation and licensed to organizations such as Mozilla Foundation.
  • Enhances web pages and can even be used to build video games.

Characteristics of JavaScript

  • Client-side scripting language (runs on the user's computer).
  • High-level scripting language (similar to English).
  • Interpreted language (converted into machine code during execution).
  • Object-oriented programming language.
  • Case-sensitive.
  • Semi-colons at the end of statements are optional.

What Can JavaScript Do?

  • React to events such as mouse clicks.
  • Validate form data.
  • Create cookies.
  • Enhance and make web pages interactive.

JavaScript Concepts

Events

Events are actions such as mouse clicks or page loading that trigger JavaScript code.

Form Validation

JavaScript checks user input in forms before the data is sent to the server, saving time and resources.

Cookies

Cookies store information on a visitor’s computer to remember preferences or monitor user activity.

Dynamic Websites

Platforms like Google, YouTube, and Facebook use JavaScript for real-time updates, streaming, and interactive features.

DHTML Effects

JavaScript combined with CSS can create effects like message scrollers, animations, and visual effects.

User Interaction

JavaScript detects user actions such as clicks, typing, or scrolling to create responsive web pages.

Advantages of JavaScript

  • Lightweight programming language.
  • Works on all modern browsers.
  • Easy to learn and beginner-friendly.
  • No special tools required.
  • Large online developer community.

Disadvantages of JavaScript

  • Some browsers disable JavaScript for security reasons.
  • JavaScript code can be viewed by users.
  • Malicious scripts may affect security.
  • Code can be reverse engineered.

Note

A script is a set of computer instructions that connects components to accomplish tasks. JavaScript is a powerful scripting language used to create web functionalities.

The main tools needed for JavaScript are:

  • A text editor
  • A web browser

The basic components of JavaScript include:

  • Objects
  • Methods
  • Properties

How to Start Using JavaScript

The <script></script> tag is used to add JavaScript in an HTML page.

Example:

<script>
document.write("Hello World");
</script>
