LESSON 4: Events



What are Events?

Events are actions that are triggered by user activities such as mouse clicks, keyboard strokes, loading images, submitting forms, or calling functions. Events are usually connected with functions because functions need something to trigger them before they can run.

Common Event Triggers

  • onclick - triggered when an object is clicked.
  • ondblclick - triggered when an object is double-clicked.
  • onkeydown - triggered when a keyboard key is pressed.
  • onkeyup - triggered when a keyboard key is released.
  • onmouseover - triggered when the mouse moves over an element.
  • onmouseout - triggered when the mouse moves away from an element.
  • onload - triggered when a page or image finishes loading.
  • onunload - triggered when the user exits the page.
  • onsubmit - triggered when a form is submitted.
  • onchange - triggered when a user changes the value of a field.

onload and onunload

The onload and onunload events are triggered when a user enters or leaves a webpage. They can also be used for checking browser versions or storing and retrieving cookies. A cookie is a message sent to a web browser for storing information.

onfocus, onblur, and onchange

These attributes are commonly used for validating form fields.

  • onfocus - triggered when an element gets focus.
  • onblur - triggered when an element loses focus.
  • onchange - triggered when the value of a field changes.

Popup Boxes

Popup boxes are windows that appear on the screen. They are used to warn users, collect information, or ask for confirmation. JavaScript has three types of popup boxes:

  • Alert Box
  • Prompt Box
  • Confirm Box

Alert Box

An alert box displays a warning or message to the user.

Example:

alert("WARNING!");

Prompt Box

A prompt box asks the user to enter information before continuing.

Example:

var name = prompt("Please enter your name:", "nameless");

Confirm Box

A confirm box asks the user to verify an action. The user can choose OK or Cancel.

Example:

confirm("Do you want to proceed?");

Variables

A variable is a temporary storage for values or expressions. Expressions are formulas that use numbers and operators.

Example:

var x = 5;
var y = x + 5;

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus
  • ++ Increment
  • -- Decrement

Comparison Operators

  • == equal to
  • === equal value and type
  • != not equal
  • !== not equal value or type
  • > greater than
  • < less than
  • >= greater than or equal
  • <= less than or equal

Logical Operators

  • && AND
  • || OR
  • ! NOT
