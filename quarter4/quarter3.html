Lesson 1: Formatting Text with CSS

This lesson introduces basic CSS text formatting such as fonts and colors.


<style>
p{
font-family: Arial;
font-size:20px;
color:purple;
font-style:italic;
}
</style>

<p>This text is formatted using CSS font properties.</p>

Lesson 1a: Font Variant


<style>
.normal{
font-variant:normal;
}

.smallcaps{
font-variant:small-caps;
}
</style>

<p class="normal">This is normal text.</p>
<p class="smallcaps">This text uses small caps.</p>

Text Shadow Example


<style>
h1{
color:white;
text-shadow:3px 3px 5px purple;
}
</style>

<h1>Text Shadow Example</h1>

Word Wrap Example


<style>
.box{
width:200px;
border:2px solid black;
word-wrap:break-word;
}
</style>

<div class="box">
ThisIsAVeryVeryLongWordThatWillBreakInsideTheBox
</div>

Lesson 2: Text Indent and Text Transform


<style>
p{
text-indent:50px;
}

.upper{
text-transform:uppercase;
}
</style>

<p class="upper">this sentence becomes uppercase</p>

</div>
