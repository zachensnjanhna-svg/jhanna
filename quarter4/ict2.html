LESSON 2: Scripts

Where Can You Place Scripts?

JavaScript codes can be placed in different sections of an HTML document depending on when they should execute. Scripts may run before the page loads, when the page loads, or when a specific event occurs.

Scripts in the HEAD

Scripts placed in the HEAD section are pre-loaded. This means the script runs before the user triggers an event.

<script type="text/javascript"> function intro() { alert("Welcome Visitor!"); } </script>

Scripts in the BODY

Scripts inside the BODY section run when the webpage loads.

<script type="text/javascript"> document.write("Message by JavaScript"); </script>

HEAD & BODY Scripts

JavaScript can be placed in both the HEAD and BODY sections depending on the needs of the program.

Head Section document.write("Head Section"); Body Section document.write("Body Section");

External JavaScript

External JavaScript files allow one script to control multiple HTML files. These files are saved with a .js extension.

<script src="myJavaScript.js"></script>

The src attribute tells the browser where the external JavaScript file is located.

Comments in JavaScript

Comments are used to explain code or temporarily disable lines during debugging.

// Single line comment /* Multiple line comment */

JavaScript Guidelines

  • Case Sensitive
  • White Space
  • Breaking Up Code Lines

Case Sensitive

JavaScript distinguishes between uppercase and lowercase letters.

Example: myHome is different from MyHome.

White Space

JavaScript ignores extra spaces but they help make code easier to read.

var name = "Grace"; var name="Grace";

Breaking Code Lines

You can break long lines using a backslash (\).

document.write("wow\Philippines"); // Accepted document.\write("wow Philippines"); // Not accepted
