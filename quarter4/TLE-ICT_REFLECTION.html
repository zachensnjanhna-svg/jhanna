Lesson 1: JavaScript

In Lesson 1, I learned that JavaScript is used to make websites interactive and dynamic. It works with HTML and CSS to add functions like responding to user actions and displaying messages. This lesson helped me understand how JavaScript makes webpages more useful and engaging.

Lesson 2: CSS Style Sheet

In Lesson 2, I learned where JavaScript scripts can be placed in an HTML document, such as in the head, body, or an external file. I also learned about JavaScript comments and some basic coding guidelines like case sensitivity and using whitespace to make code easier to read. This lesson helped me understand how to properly organize JavaScript in a webpage.

Lesson 3: Characteristics of JavaScript

In Lesson 3, I learned about the Characteristics of JavaScript, and also about JavaScript variables and how they are used to store data in a program. Variables help keep information that can be used or changed while the program runs. This lesson helped me understand how data is handled in JavaScript and how it is important for creating interactive web pages.

Lesson 4: Events

In Lesson 4, I learned about JavaScript operators and how they are used to perform operations on variables and values. These include arithmetic operators for calculations and comparison operators for checking conditions. This lesson helped me understand how JavaScript processes data and performs different tasks in a program.
