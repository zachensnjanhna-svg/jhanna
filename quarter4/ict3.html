LESSON 3: Characteristics of JavaScript

Characteristics of JavaScript

1. Interpreted Language - JavaScript requires a browser to run. It works together with HTML and its source code can easily be viewed when checking the source code of a webpage.

2. Scripting Language - JavaScript is used to perform simple repetitive tasks and modify parts of a webpage through the browser.

3. Client-Side Scripting - JavaScript runs on the user's computer (client) rather than the server.

4. Object-Oriented Programming - JavaScript treats many parts of the webpage as objects that can be manipulated using methods and properties.

The Key in Understanding JavaScript

  • Learn at an optimal pace and practice frequently.
  • Understand the concept of objects.

Objects - items that exist in the browser such as the browser window or webpage.

Methods - actions performed by an object.

Properties - characteristics of an object.

Event - an action such as clicking a link or loading a page.

Parameter - information passed to a function.

Type - the kind of data stored in a variable.

Type Conversion - converting one type of data to another.

Commonly Used Properties and Methods

  • Window - the main JavaScript object.
  • location - used to go to another webpage.
  • open() - opens a new browser window.
  • setTimeout() - runs an action after a time delay.
  • document.write() - displays a message on a webpage.
  • lastModified - shows the last modified date of a page.
  • Math.PI - value of PI.
  • Math.max() - returns the largest number.
  • Math.min() - returns the smallest number.
  • navigator.appName - browser name.
  • navigator.appVersion - browser version.
  • navigator.platform - operating system.

Example JavaScript Code

document.write("Hello World");

This code tells JavaScript to display the message 'Hello World' on the webpage.

JavaScript Variables

A variable is a container that stores a value which can change during program execution.

var firstname = "Crispin";

Data Types

  • Character
  • String
  • Integer
  • Float
  • Boolean
  • Function
  • Object
  • Array
  • Undefined
  • Null

Rules for Naming Variables

  • Use letters, numbers, and underscores.
  • No spaces are allowed.
  • The first character cannot be a number.
  • Variable names are case-sensitive.
