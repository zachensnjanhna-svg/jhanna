Lesson 6: Setting Dimensions

Property for Dimensions

CSS can be used to manage dimensions such as visibility, width, height, and more.

Some common properties include:

  • visibility: Controls if an element is visible or hidden.
  • width: Specifies the width of an element.
  • height: Specifies the height of an element.
  • line-height: Controls the spacing between lines of text.
  • max-width: Limits the maximum width of an element.
  • min-height: Specifies the minimum height of an element.

Pseudo-Classes and Links

Pseudo-classes are used to define the state of an element, such as when a user interacts with it.

Some common pseudo-classes for links:

  • :link - Applied to unvisited links.
  • :visited - Applied to visited links.
  • :hover - Applied when the mouse hovers over the element.
  • :active - Applied when the element is clicked.
