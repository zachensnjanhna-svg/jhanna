Lesson 4: CSS Rule & Classes

CSS Rule

A CSS rule has two main parts: a selector, and one or more style declarations.

The basic format of a CSS style rule is as follows:

For Inline style rule:



For document-level and external style sheets:
selector {style declarations}
selector {prop1:value; prop2:value; prop3:value;}

Style Properties

Properties in CSS are collected into six groups:

  • Fonts
  • Colors and backgrounds
  • Text
  • Boxes and layout
  • Lists
  • Tag classifications

Classes

What are Classes?

A class is a user-defined selector that allows you to apply a style rule to specific HTML tags.

Classes are reusable, meaning you can apply the same set of formatting rules to multiple HTML elements. You define a class using a dot (.) followed by the class name.
