CSS Syntax

Structure of CSS

Selector

The HTML element you want to style; is a pattern used to "find" or "select" the HTML elements on a webpage to which a set of CSS rules will be applied.

Property

The style attribute you want to modify; is a named characteristic or attribute that defines a specific aspect of how an HTML element should be displayed or rendered by the browser.

The Style Sheet and its Parts

A style sheet is a set of instructions to a web browser on how to display various elements on a web page.

Every CSS (whether it is contained in a .css file, or embedded in the head element of an HTML document) is a series of instructions called statements.

How Do Style Sheets Work?

Style sheets are just text files, or text embedded in the head of an HTML document, that help separate content from appearance.

The content of the page goes into an HTML file, and the appearance goes into a style sheet.

A style sheet suggests the browser how to display a page in a particular way.

Three Kinds of CSS

External Style Sheets

External style sheets allow you to develop a consistent style across pages and change the layout of your entire website by simply changing one CSS file.

Embedded Style Sheets

Embedded style sheets are placed within the <head> and </head> tags and override external style sheets.

Inline Style Sheets

Inline style sheets are used for isolated changes to a headline, paragraph, or other element and are placed directly in the style attribute of an HTML tag.

Types of CSS Example Codes

CSS Comments

CSS comments are used to explain the code and help you edit it later. Comments are ignored by the browsers.
