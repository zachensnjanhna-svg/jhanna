Lesson 3: Display Property

What is the Display Property?

The CSS display property allows you to control how an element is displayed on a web page. It determines the behavior of an element in terms of its layout and positioning.

Kinds of Elements in CSS

There are different kinds of elements in CSS that affect how content is displayed:

  • Block-level elements: These elements take up as much space as possible and will start a new line on the page. Examples include:
    • <p> (Paragraph)
    • <ol>, <ul>, <dl> (Lists)
    • All headings (<h1>, <h2>, etc.)
    • <article>, <section>, <div> (Containers)
  • Inline-level elements: These elements are laid out horizontally and do not force the content after them to a new line. Examples include:
    • <a> (Anchor/Link)
    • <strong>, <em>, <b>, <i> (Text styling)
    • <span> (Generic inline container)

Why is This Important?

Understanding how an element behaves when styled with CSS is crucial because the display mode of an element (block vs. inline) affects how it interacts with other elements on the page. Some CSS properties behave differently depending on whether the element is block-level or inline.

For example, inline elements cannot contain block-level elements, as demonstrated below:

<em> ? Incorrect
<p>An emphasized paragraph.</p>
<p>Another emphasized paragraph.</p>
</em> ? Incorrect

The correct way to structure this is to place the inline element inside the block elements:

<p><em>An emphasized paragraph.</em></p>
<p><em>Another emphasized paragraph.</em></p>

Property for Display

The display property in CSS can take several values, each affecting the element's layout differently:

  • display: inline; - The element is laid out horizontally (inline layout).
  • display: block; - The element is laid out vertically (block layout).
  • display: list-item; - The element is displayed as a list item (<li>).
  • display: none; - The element is removed from the layout entirely and has no effect on the page.
  • display: inherit; - The element inherits the display value from its parent element.

Div and Span Tags

The <div> and <span> tags are used to group content together, but they behave differently:

  • <div>: A block-level element used to divide the content into sections. It is often used for layout and grouping.
  • <span>: An inline element used to style or structure a part of content without affecting the flow.
