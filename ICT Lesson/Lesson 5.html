Lesson 5: CSS Selectors

CSS Selectors

CSS selectors are used to find or select the HTML elements that you want to style.

CSS selectors are divided into 6 types:

  • Class selector
  • Element selector or Tag selector
  • ID selector
  • Universal selector
  • Group selector
  • Attribute selector

Element Selector

The element selector selects HTML elements (p, div, h1, etc.) and applies CSS to them.

ID Selector

The ID selector is used to style one unique element in a page.

Universal Selector

The universal selector selects every single HTML element on the page. It is written using the asterisk (*) character.

Group Selector

The group selector allows you to select multiple elements and apply the same style to all of them.

Attribute Selector

The attribute selector selects elements based on specific attribute values.

Example

Element Selector

Applies styles to HTML elements, like p {color: red;}

ID Selector

Applies styles to elements with a specific ID, like #header {color: blue;}

Class Selector

Applies styles to elements with a specific class, like .highlight {background: yellow;}
